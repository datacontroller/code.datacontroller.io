Detailed Description

STP for creating the configuration tables in DEMO mode. It also adds the STAGING directory as subdirectory to the BASE library location. Note - the CURLIB var is added during the build process.

Warning
This STP self destructs! It will delete itself after a successful run to avoid being executed twice (and overwriting actual data)
